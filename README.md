## 📖 關於

個人部落格，記錄關於**系統架構、認知科學、存在主義**與**獨立思考**的文字。

這裡沒有聳動的標題，也沒有急著說服的論點。只是一個人在深夜寫下來的東西——關於那些反覆出現卻始終沒有答案的問題。

```bash

pnpm dev

# 建置
pnpm build

# 預覽建置結果
pnpm preview
```
